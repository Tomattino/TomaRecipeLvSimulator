# 🍅 料理レベ上げシミュ（アルファ版）

ポケモンスリープの**料理レベル上げに必要な調理回数**をシミュレーションするWebツールです。

> ⚠️ **アルファ版です。** 作りたい機能をベースに開発中のため、動作の保証はしません。予告なく仕様変更・機能追加が発生することがあります。

---

## 機能

料理のレベル上げシミュレーションができます。

---

## 技術スタック

- [Vue 3](https://vuejs.org/) (Composition API)
- [Pinia](https://pinia.vuejs.org/)
- [Vite](https://vitejs.dev/)
